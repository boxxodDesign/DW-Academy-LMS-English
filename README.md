# DW Academy LMS - English Version

Premium sales training platform for DW Drums sellers, now fully translated to English and branded with official identity.

## Features
- Full course management for Admins.
- Interactive course player with video and text.
- Trivia system for knowledge assessment.
- Salesperson, Instructor, and Admin dashboards.
- Premium DW branding and UI.

## Local Development
1. `npm install`
2. `npm run dev`
